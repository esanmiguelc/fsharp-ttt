# Tic tac toe in F#

## Setting up

  1. Make sure you have [dotnet core](https://www.microsoft.com/net/core) installed
  2. Clone the repo 

    $ git clone git@github.com:esanmiguelc/fsharp-ttt.git

  3. Fetch nuget dependencies

    $ dotnet restore

## Running the tests

    $ dotnet test Core.Tests/Core.Tests.fsproj
